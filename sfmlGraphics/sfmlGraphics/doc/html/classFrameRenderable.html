Detailed Description

This class is the first renderable you will use (see the Tutorial 03 of practical 01). It will render the world frame on the screen: a red line for the X axis, a green line for the Y axis and a blue line for the Z axis. Have a look at the source: we documented it to help you to understand the details.

Member Data Documentation

unsigned int FrameRenderable::m_cBuffer
private

Store the location (index) of the color attribute buffer on the shader program used to render the instance. If all instances are rendered by the same shader program, you could make this location static and set this location when the first instance is created. We decided to not make it static as it would be harder to understand and it is quite constraining: if you want to use more than shader program to render frame renderable instances, this will lead to bugs tricky to understand for beginners.
