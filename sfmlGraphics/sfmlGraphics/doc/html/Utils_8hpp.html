const float &  low,
const float &  high

This function allows to visually represent one dimensional parameter thanks to a color code: cold colors (blue) for small values and hot colors (red) for big values.

Parameters
factorThe factor to visually represent.
lowThe minimum expected value for the factor (will return a blue color).
highThe maximum expected value for the factor (will return a red color).
Returns
The generated color vector.
