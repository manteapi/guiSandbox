Detailed Description

The Viewer class allows to visualize virtual contents and interact with it. It is the key class of the the programs you will build in these practicals.

Member Typedef Documentation

typedef std::chrono::system_clock Viewer::clock

We use the system_clock of std::chrono to compute durations. chrono is a nice addition to the c++ std, go have a look there: http://www.cplusplus.com/reference/chrono/

typedef std::chrono::duration<double> Viewer::Duration

We use a double precision duration type to have a precise animation.

typedef std::chrono::time_point< clock, Duration > Viewer::TimePoint

Based on the clock and the duration types, we define a time point type.
