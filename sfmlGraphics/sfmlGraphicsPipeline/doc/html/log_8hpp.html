)    LOG_WITH_LINE_FILE( level, message, __LINE__, __FILE__ )

Log an entry with a specific severity_level. For levels above severity_level::warning, the location (file and line) of the code that request to log that entry is automatically included in the entry. The message can be built by concatening strings and values with the stream operator:

float x = 2, y = 5;
LOG( debug, "the 2D point is at {" << x << ", " << y << "}");

Enumeration Type Documentation

enum severity_level

A severity level refers to a particular kind of information logged in a log entry by the macro LOG(level,message). Its value reflects its importance, from trace which is the less important to fatal which is critical.

Enumerator
trace 

the log entry is a trace to understand the program flow

debug 

the log entry is a debug information, that should not appear in production code

info 

the log entry is an information that might interest the user

warning 

the log entry is a warning about something that is not an error but might not be what the user expected

error 

the log entry is an error, something needs to be fixed

fatal 

the log entry is a fatal error, terminating immediately the application
